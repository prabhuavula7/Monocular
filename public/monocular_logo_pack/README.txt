# Monocular Logo Pack

Transparent-background SVG logo kit for **Monocular**.

## Included files

- `monocular-icon-color.svg` - Primary icon mark
- `monocular-icon-black.svg` - Icon mark for light backgrounds
- `monocular-icon-white.svg` - Icon mark for dark backgrounds
- `monocular-app-icon.svg` - Large square app icon
- `monocular-horizontal-black.svg` - Full logo with black wordmark
- `monocular-horizontal-white.svg` - Full logo with white wordmark
- `monocular-horizontal-monochrome-black.svg` - Full monochrome black logo
- `monocular-horizontal-monochrome-white.svg` - Full monochrome white logo
- `monocular-stacked-black.svg` - Stacked lockup with black text
- `monocular-stacked-white.svg` - Stacked lockup with white text
- `monocular-wordmark-black.svg` - Text-only black wordmark
- `monocular-wordmark-white.svg` - Text-only white wordmark
- `monocular-mark-micro.svg` - Small simplified mark for favicon/social/avatar use

## Palette

- Orange: #F28C28
- Black: #0B0B0D
- Purple: #5F2A8A
- White: #FFFFFF
- Gold: #C9A227

All assets have transparent backgrounds.
